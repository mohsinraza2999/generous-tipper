# End-to-End Generous Tip Giver Prediction

## Problem
Predict generous tip giver customers for a New York Taxi Driver.

## Solution
Built a full ML pipeline including:
- Data ingestion & cleaning
- Feature engineering
- Model training (XGBoost, Random Forest, Logistic Regression)
- REST API deployment
- Dockerized application

## Tech Stack
Python, Pandas, Scikit-learn, FastAPI, Docker
